# 리스트 자료형

```
리스트명 = [요소1, 요소2, 요소3...]
list = [1, 2, 3, 4, 5...]
```

1. 비어있는 리스트 생성
```python
a = list()
```

2. 리스트 인덱싱
```python
a = [1, 2, 3]

a[0] 
# 1
a[0] + a[2]
# 4

a = [1, 2, 3, ['a', 'b', 'c']]

a[3]
# ['a', 'b', 'c']

# 마지막 요소는 -1의 인덱스로도 값을 찾을 수 있음
a[-1]
# ['a', 'b', 'c']
a[-1][1]
# 'a'
```
